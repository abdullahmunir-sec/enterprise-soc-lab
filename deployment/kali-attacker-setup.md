# Kali Linux Attacker Machine

A Kali Linux virtual machine was deployed to simulate an external attacker within a controlled network segment.

## Configuration

- OS: Kali Linux
- Network: Attacker segment (192.168.3.0/24)
- Purpose: Launch simulated attacks against internal assets and generate malicious traffic for SOC detection.

## Usage in the Lab

This system is used to:
- Perform network scanning and enumeration
- Launch exploitation attempts against vulnerable services
- Simulate real-world attacker techniques in a safe environment
